This plugin allow you to insert in your posts or pages years stats, tot nr. of post,tot nr. of comments and some average.
== Description ==

With this plugin you can publish years stats about your blog:

*Nr. Post
*Avg post character
*Tot post character
*Avg Comments
*Tot Comments

If you want to contribute to this project here is the github repository: https://github.com/wolly66/wpit-blog-stats


== Installation ==

1. Upload `Wpit Blog Stats` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create stats and insert the right shortcode in a page or post.

== Frequently Asked Questions ==
* Why use this plugin?

To show stats of your website

* How To show stats in posts or pages

Insert the shortcode or shortcodes in a page or in a post.

* Where I found the shortcode?

In backend, under Settings menu you will find the Wpit Stat Settings menu.
